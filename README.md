# 📝 Task Manager – Full Stack Web Application

##  Overview
Task Manager is a full-stack web application designed to help users efficiently manage their tasks, track progress, and collaborate with others. The platform allows users to create, update, and organize tasks while supporting real-time interactions and structured workflow management.

---

## 🎯 Problem Statement
Managing tasks across teams or individually can become complex without a centralized system. This project solves that problem by providing a structured platform where users can create tasks, track deadlines, collaborate with others, and manage progress effectively.

---

## 🧠 Features

- 🔐 User Authentication (Login/Signup)
- 📋 Create, Update, Delete Tasks
- ⏳ Deadline Tracking (Upcoming, Completed, Overdue)
- 👥 Collaboration (Multiple users per task)
- 💬 Messaging System (Task-level communication)
- 📂 File Upload & Management
- 🔄 Task Status Updates (Complete / Undo)
- 📊 Organized Dashboard View

---

## 🏗️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js / Express *(or your actual backend if different)*
- **Database:** MongoDB / SQL *(update based on your project)*
- **Other Tools:** REST APIs

---

## ⚙️ How It Works

1. User logs in to the platform
2. Creates a new task with deadline and collaborators
3. Tasks are categorized into:
   - Upcoming
   - Completed
   - Overdue
4. Users can:
   - Edit task details
   - Upload files
   - Send messages
5. Tasks automatically move based on deadline status
